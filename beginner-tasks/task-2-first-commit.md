# Task 2: Make Your First Commit

## 🎯 Learning Goal
Learn what a "commit" is and make one yourself.

## 📖 What's a Commit?

A **commit** is like saving a game checkpoint. It captures:
- What changed
- Who changed it
- When it changed
- Why it changed (your message)
