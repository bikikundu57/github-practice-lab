# Task 1: What is Git and GitHub?

## 🎯 Learning Goal
Understand the difference between Git and GitHub.

## 📖 Quick Explanation

- **Git** = The tool that tracks changes (runs on your computer)
- **GitHub** = The website where you save/share your Git projects

Think of Git like a **time machine** for your files.  
Think of GitHub like **Google Drive** for coders.
