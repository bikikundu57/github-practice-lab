# Task 4: Working with Branches

## 🎯 Learning Goal
Learn what branches are and how to create, switch, and merge them.

## 📖 What is a Branch?
A **branch** is like a parallel universe for your code. You can experiment without breaking the main version.

## 📝 Steps

### Step 1: Create a Branch (GitHub Website)

1. Go to your repository on GitHub
2. Click the **"main"** dropdown button (top left)
3. Type a branch name: `add-my-name`
4. Click **"Create branch: add-my-name"**

### Step 2: Make Changes on Your Branch

1. Make sure you're on `add-my-name` branch (it shows at top)
2. Click **"Add file"** → **"Create new file"**
3. Name: `my-info.md`
4. Content:
   ```markdown
   # About Me
   
   - Name: [Your Name]
   - Learning: Git and GitHub
